## 核心贡献
用角度+距离混合测量解决多智能体编队缩放(obstacle avoidance)问题。
- 角度控制形状，距离控制缩放
- 只需1个agent测距(其余agent只需测角)，大幅减少传感负担
- 放松了angle rigidity条件，用formation rigidity构造正方形

## 关键假设 (Assumptions 1-5, inline, no subsections)
Assumption 1: All agents already on the same plane.
Assumption 2: Each agent equipped with LOS sensor.
Assumption 3: Initial positions independent and non-collinear.
Assumption 4: Agent #3 measures distances to other agents. Others don't need distance.
Assumption 5: (Polygonal expansion) Desired positions of new agent and 3 neighbors not on a circle; two measured angles not zero or π.

## 关键定义
- Interior angle: α_i ∈ [0, 2π), measured in local frame through LOS bearings
  cos α_i = z_ij^T z_ik, where z_ij = (p_j - p_i)/‖p_j - p_i‖
- Angle set A = {(j,i,k)} encoding shape constraints

## 三角形编队控制 (3.1)
Desired: 3 angles α_i* + 1 distance d13*
Control law for agent i=1,2,3:
  u_i = v̇_c - k1(v_i - v_c) - k2(α_i - α_i*)(z_i(i-1) + z_i(i+1))
  u_3 additionally: + k3(d13 - d13*)(z31 + z32)

关键结构: z_ij + z_ik 指向内角平分线方向(angle bisector)
- 角度误差修正沿角平分线方向
- 距离误差修正也沿角平分线(agent #3)

Stability: 构造9维状态 X = [d13-d13*, e1, e2, v1-vc, v2-vc, v3-vc]
利用 e3 = -(e1+e2) (三角形内角和=π) 降维
线性化矩阵具有块结构 → Hurwitz via Lemma 1

## 多边形编队控制 (3.2)
扩展到n个agent，每个agent i>3测量相对于3个邻居的两个角度
角度集 A = {(j1,i,j2), (j2,i,j3)}，其中 j1,j2,j3 < i
控制律形式同三角形，但 k3=0 for i≠3
结构矩阵 S1^T S2 是块下三角，每个对角块 Hurwitz

Remark 4: 解释了为什么单条边距变化能决定整个编队缩放
  → 正弦定理: d23/sinα1 = d13/sinα2 = d12/sinα3 = 2R
  → 确定角度后，外接圆半径R仅与弦长相关

## 正方形编队控制 (3.3) — 对你论文最相关！
### 问题: 四点在圆上的正方形
- 四个内角都是π/2 不满足angle rigidity（圆上任意四点内角和=2π，π/2是自然的）
- Agent #4沿弧143移动时，∠142和∠243保持π/4（非唯一性！）
- 这直接对应你论文中的reflection ambiguity问题

### 解决方案
改变约束: agent #3测量d32和d34（两条边），而非之前的一条d13
控制律:
  u_i = v̇_c - k1(v_i - v_c) - k2(α_i - π/2)(z_i(i-1) + z_i(i+1)), i∈{1,2,4}
  u_3 = v̇_c - k1(v_3 - v_c) - k2(α_3 - π/2)(z32 + z34)
        + k3(d32 - d32*)z32 + k3(d34 - d34*)z34

关键区别(Remark 5): 距离误差权重变了
- 三角/多边形: 权重均匀分配在角平分线方向
- 正方形: 每条边的距离误差只沿该边方向, e_{d32}z32 + e_{d34}z34 (非均匀权重)

13维状态 → 线性化 → Hurwitz

## QUAV 应用 (Section 4)
使用简化QUAV模型(与你的论文一致):
  ṗ_h = v_h, v̇_h = -g A_ψ Θ_h    (水平通道)
  ṗ_z = v_z, v̇_z = g - f/m        (高度通道)
  A_ψ = R_ψ [[0,1],[-1,0]], Θ_h = [φ, θ]^T
  φ∈(-π/2,π/2), θ∈(-π/2,π/2), ψ∈[-π,π)  ← 注意姿态角约束！
控制结构: 外环位置(FORMATION) → 内环姿态(PD)
仿真场景: 遇狭窄地形时缩小d13穿过，通过后恢复
