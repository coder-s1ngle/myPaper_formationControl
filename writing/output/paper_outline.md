# 论文大纲草稿
## 一、摘要

摘要建议按下面 5 句话展开：

1. 先交代问题背景：面向四智能体无向平面方形编队控制，考虑当前实现中包含角度、距离、共面、法向和速度项的闭环控制结构。
2. 说明本文研究对象：为得到严格可证明结论，本文在固定分支、固定常值参考法向、无饱和控制、LESO 局部精确补偿绳索张力的口径下，研究冻结名义外环模型。
3. 说明方法：构造最小内部误差坐标，证明一个角误差局部冗余，建立最小内部误差闭环系统，并独立推导其线性化结构矩阵与特征多项式。
4. 说明结果：证明代表性目标方形附近的最小内部误差系统局部指数稳定，并将结论推广到同一固定分支上的任意同构目标方形。
5. 说明边界：指出当 LESO 补偿残差不为零时，更合适的结论应理解为局部 ISS、局部一致最终有界或局部 practical stability。

如果按“反无人机捕网”背景重写摘要，第一句更建议改成：

针对多无人机协同张网拦截来袭小型无人机时的编队稳定保持问题，本文研究四智能体无向平面方形编队控制律在目标捕网构型附近的局部稳定性。

## 二、引言

### 2.1 研究背景

这一段建议写：

- 反无人机场景下，单机拦截往往存在目标机动性强、接触风险高和拦截包络有限等问题，多无人机协同张网捕获是一类更安全的物理处置方案。
- 在这类任务中，四机方形编队可以作为网口支撑构型，编队不仅要保持边长和局部角关系，还要保持共面性与法向姿态，以便在接触瞬间形成有效包络。
- 捕网过程中常伴随目标偏心撞网、绳索张力突变和瞬时外部扰动，因此仅讨论理想几何构型是不够的，还需要分析具体闭环控制律在目标构型附近的稳定性。
- 相比依赖全局位姿或严格坐标对齐的方法，角度与局部几何信息更容易由机载视觉或局部感知获得，也更贴近捕网任务的工程实现。

### 2.2 场景化问题引出

建议用一个短小节把任务和理论对象连起来：

- 实际任务是四架无人机牵引柔性网，对目标无人机实施包络式捕获。
- 对控制层而言，核心问题可以抽象为：在存在绳索张力、外部冲击和法向分支选择的情况下，如何保持方形网口编队在目标平面附近稳定。
- 因而，本文并不直接研究捕获接触后的完整混杂过程，而是聚焦于捕获前后一个局部工作区间内的编队稳定保持问题。

### 2.3 相关工作

1. 角刚性与二维角度编队控制。
2. 仅用角度测量的三角形、多边形、方形构型控制。
3. 面向卫星编队的角度测量控制。
4. 引入少量距离信息处理尺度问题的局部信息编队控制。

这一节可以直接吸收 `related_work_contribution.md` 里的对比内容。

如果你希望更贴合应用背景，可以在这节开头加一句：

### 2.4 现有工作的不足

这一小节建议明确压出 4 个缺口：

1. 现有结果多集中于二维几何构型或单积分模型，与当前双积分四智能体方形平面编队控制律并不一致。
2. 现有结果通常不直接分析“角度 + 距离 + 共面 + 法向 + 速度”耦合闭环。
3. 很少显式构造与系统对称性匹配的最小内部误差坐标，也缺少对冗余角误差的严格处理。
4. 理论模型与实际实现之间，尤其是 LESO 张力补偿和无向分支逻辑之间，往往缺少清晰桥接。

如果按捕网背景表述，可以把第 4 点改得更场景化一些：

4. 面向捕网任务中常见的绳索张力变化与目标冲击扰动，理论模型与实际实现之间仍缺少清晰的桥接分析。

### 2.5 本文贡献

这一节建议直接列 4 点：

1. 针对反无人机捕网中的四机方形网口保持问题，构造与系统对称性匹配的最小内部误差坐标，并证明原始误差中的一个角误差在局部上是冗余变量。
2. 在固定分支、固定常值法向和无饱和控制的名义模型下，独立推导闭环误差系统线性化矩阵与特征多项式，证明最小内部误差系统局部指数稳定。
3. 基于刚体共轭关系，将结论推广到同一固定分支上的任意同构目标方形，得到坐标无关的局部指数稳定性，从而覆盖同一类网口目标构型。
4. 结合捕网任务中的绳索张力补偿与目标冲击扰动，给出理论与实际实现之间的桥接说明，并说明 LESO 补偿残差不为零时的 ISS 口径。

### 2.6 文章结构
