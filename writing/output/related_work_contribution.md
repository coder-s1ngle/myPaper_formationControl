# related paper 对应的本文 contribution 草稿

## 1. 相对 *Angle Rigidity and Its Usage to Stabilize Multiagent Formations in 2-D* (2021)

这篇工作建立了二维角刚性理论，并说明仅靠角度约束可以刻画一类编队形状，其重点是角刚性的定义、判定条件以及二维角度编队控制。相较之下，本文并不再从一般性的角刚性理论出发，而是面向当前四智能体方形编队控制律，直接构造与该控制器匹配的最小内部误差系统。本文的贡献在于：在三维无向平面编队场景下，把角误差、距离误差、共面误差、法向误差和速度误差统一纳入同一闭环模型，证明原始误差中存在一个局部冗余角变量，并进一步给出最小内部误差系统的局部指数稳定性结论。也就是说，本文关注的不是“哪些角约束可以决定形状”，而是“当前这套具体控制律在目标方形附近为什么稳定，以及稳定到什么程度”。

## 2. 相对 *Multi-agent formation control using angle measurements* (2019)

这篇工作主要讨论二维三角形与凸多边形编队，只使用角度测量，通过几何构造和 Lyapunov 方法证明闭环稳定。本文与其最主要的区别有两点。第一，本文处理的不是二维运动学模型，而是带速度状态的双积分型四智能体编队，并且目标构型是三维空间中的方形平面编队。第二，本文控制目标不再只是角度收敛，而是同时约束边长、共面性、参考法向和速度，从而得到一个更接近实际实现的闭环误差模型。基于此，本文进一步给出了最小内部误差坐标、显式线性化结构矩阵以及 Hurwitz 判定，而不是停留在几何构造意义下的稳定性说明。

## 3. 相对 *Satellite Formation Flying Control by Using Only Angle Measurements* (2023)

这篇工作已经把角度测量编队方法推进到了卫星编队背景，并讨论了三角形、多边形以及方形构型的构造问题，是和本文最接近的一篇相关工作。相比之下，本文的贡献不在于提出一套新的“只用角度测量”的构型设计方法，而在于对当前四智能体方形控制律给出更细的稳定性刻画。具体来说，本文围绕固定分支、固定常值参考法向这一工作区间，构造了 8 维最小内部误差坐标，证明了一个角误差在局部上可以由其余误差光滑恢复，并据此把原始 21 维闭环系统规约为 20 维最小误差系统。随后，本文独立推导出线性化矩阵及其特征多项式，证明最小误差系统在目标方形附近局部指数稳定。除此之外，本文还把实际实现中的 LESO 张力补偿和无向法向迟滞分支逻辑纳入分析口径，说明理论结论在什么条件下可以作用到实际控制实现。

## 4. 相对 *Multi-agent distributed formation control using local information measurements* (2025)

这篇工作进一步考虑了局部信息下的编队变形与尺度调节，通过“角度 + 少量距离”来处理纯角度方法中的尺度模糊问题，并在双积分模型下给出局部收敛分析。本文与其关系更适合表述为互补，而不是简单替代。该文的重点是“如何用尽量少的局部距离信息实现尺度调节”，而本文的重点是“在固定尺寸、固定分支、固定参考法向的方形编队附近，如何把当前控制律的最小内部误差系统完整地证明为局部指数稳定”。因此，本文的贡献主要体现在两个方面：一是给出比一般局部收敛更强的局部指数稳定结论；二是把最小误差坐标、冗余角消去、结构矩阵显式分解、坐标无关推广，以及 LESO 补偿残差下的 ISS 口径统一到同一套分析框架中。

## 5. 可直接写进论文大纲的 contribution

如果你想在大纲或引言里用比较正式的写法，可以直接写成下面这几条：

1. 本文针对四智能体方形无向平面编队的当前控制律，构造了与系统对称性相匹配的最小内部误差坐标，并证明原始误差中的一个角误差在目标构型附近是局部冗余的。

2. 在固定分支、固定常值参考法向和无饱和控制的名义模型下，本文独立推导了闭环误差系统的线性化结构矩阵与特征多项式，证明最小内部误差系统在代表性目标方形附近局部指数稳定。

3. 基于刚体共轭关系，本文将上述结论从代表性目标方形推广到同一固定分支上的任意同构目标方形，得到坐标无关的局部指数稳定性结论。

4. 针对实际实现中存在的绳索张力补偿和无向法向分支切换，本文进一步给出了理论与实现之间的桥接说明：当 LESO 在局部邻域内精确补偿绳索张力且分支不发生切换时，实际外环可退化为本文分析的冻结名义模型；当补偿残差不为零时，相应结论应理解为局部 ISS、局部一致最终有界或局部 practical stability。

## 6. 一段更适合放在 introduction 里的总述

与现有角度测量编队控制工作相比，本文并不主要关注新的构型生成方法或尺度调节机制，而是面向当前四智能体方形编队控制律，建立了一套与实际实现相衔接的最小内部误差稳定性分析框架。该框架一方面揭示了原始角误差变量中的局部冗余结构，另一方面给出了固定分支、固定常值参考法向条件下最小闭环误差系统的局部指数稳定性证明，并进一步把结论推广到同一分支上的任意同构目标方形。结合 LESO 张力补偿与无向分支逻辑的局部说明，本文也明确了理论结果能够作用于实际实现的条件边界。
